================================================================================
                     자율주행 시스템 실행 순서 안내서
================================================================================

본 문서는 ROS2 Humble 환경에서 Xycar 자율주행 시스템을 실행하기 위한
절차를 설명합니다.


--------------------------------------------------------------------------------
1단계. 필수 패키지 설치
--------------------------------------------------------------------------------

아래 명령어를 터미널에서 실행하여 필요한 ROS2 패키지를 설치합니다.

    sudo apt update
    sudo apt install ros-humble-navigation2 \
                     ros-humble-nav2-bringup \
                     ros-humble-slam-toolbox \
                     ros-humble-tf2-ros \
                     ros-humble-tf2-tools


--------------------------------------------------------------------------------
2단계. 워크스페이스 빌드
--------------------------------------------------------------------------------

워크스페이스 루트 디렉터리(/home/user/xycar_ws)에서 아래 명령어를 실행합니다.

    colcon build --symlink-install && source install/setup.bash

※ 빌드 후에는 반드시 source 명령어를 실행하여 환경 변수를 갱신하여야 합니다.


--------------------------------------------------------------------------------
3단계. 유니티(Unity) 시뮬레이터 실행
--------------------------------------------------------------------------------

ROS2 launch 실행 전에 Unity 시뮬레이터를 먼저 구동하여야 합니다.
시뮬레이터가 실행 중이지 않으면 카메라, LiDAR, IMU 토픽이 발행되지 않아
시스템이 정상적으로 동작하지 않습니다.


--------------------------------------------------------------------------------
4단계. 자율주행 시스템 실행
--------------------------------------------------------------------------------

아래 명령어 하나로 전체 시스템이 실행됩니다.

    ros2 launch track_drive bringup.launch.py

위 명령어를 실행하면 다음 노드들이 자동으로 구동됩니다.

  - dead_reckoning      : IMU 및 모터 신호 기반 Dead Reckoning 오도메트리
  - lidar_translator    : LiDAR 더미값 필터링
  - motor_translator    : Nav2 cmd_vel → XycarMotor 변환
  - static_transform    : base_link → lidar_frame 좌표 변환 발행
  - Nav2 (navigation2)  : 경로 계획 및 장애물 회피
  - track_drive         : 카메라 기반 차선 추종
  - traffic_light       : 신호등 인식 및 교차로 제어
  - goal_sender         : Nav2 목표 지점 자동 전송


--------------------------------------------------------------------------------
5단계. 시스템 동작 흐름
--------------------------------------------------------------------------------

실행 후 시스템은 아래 순서로 자동 동작합니다.

  (1) 시스템 기동 후 약 15초간 Nav2 초기화를 대기합니다.
  (2) 초록 신호 감지 후 Nav2 기반 LiDAR 자율주행을 시작합니다.
  (3) Nav2 목표 지점 도달 시 motor_translator가 자동 종료됩니다.
  (4) 이후 카메라 기반 차선 추종 주행으로 자동 전환됩니다.


--------------------------------------------------------------------------------
주의사항
--------------------------------------------------------------------------------

  - ROS2 환경 변수가 설정되어 있지 않은 경우 아래 명령어를 먼저 실행하십시오.
        source /opt/ros/humble/setup.bash

  - 빌드 후 source install/setup.bash를 실행하지 않으면 이전 버전의 노드가
    실행될 수 있으므로 반드시 실행하여 주시기 바랍니다.

  - 카메라 영상(/usb_cam/image_raw/front)이 수신되지 않는 경우
    Unity 시뮬레이터를 재시작하면 해결되는 경우가 많습니다.
